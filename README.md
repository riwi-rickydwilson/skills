# Skills Repository

This repository houses custom skills for Claude Code. Skills are specialized capabilities that extend Claude Code's functionality for specific tasks and domains.

## Structure

```
skills/
├── README.md           # This file
├── examples/           # Example skills to learn from
└── custom/             # Your custom skills
```

## What are Skills?

Skills are reusable, specialized prompts and workflows that Claude Code can invoke to perform specific tasks more effectively. They provide:

- Domain-specific knowledge and expertise
- Consistent workflows for common tasks
- Specialized capabilities beyond the base system

## Creating Skills

To create a new skill:

1. Add your skill definition to the `custom/` directory
2. Follow the skill format and conventions
3. Test your skill thoroughly
4. Document its usage and capabilities

## Using Skills

Skills can be invoked using the Skill tool when available. Check the Claude Code documentation for details on skill invocation and management.

## Contributing

Feel free to add your own custom skills and share examples that might be useful for others.
